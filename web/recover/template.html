ShardQuorum Recovery

Offline recovery tool. Reconstructs a secret from a quorum of shards.

Self-test

Built-in vectors prove this tool recovers correctly. It runs on load.

How to use this, and how it works

Enter at least the quorum (K) of your shards, one per line, as either the printed ByteWords or the ur:sskr/ string. In encrypted-envelope mode, also paste the ur:sq-env/ envelope. Press Recover.

This file is self-contained: all logic runs on your device, and it never makes a network request. For a real recovery, disconnect from networks first and verify this file's checksum against the value stored with your shards.

The algorithm is specified in full in docs/RECOVERY-SPEC.md. This tool is one implementation of it; the specification plus its test vectors are the root artifact, so the secret stays recoverable even if this file is lost.
